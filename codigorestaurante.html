Código Parcial y Funcionamiento

Fragmentos clave del sistema POS para restaurantes desarrollado y optimizado para uso rápido, delivery, pagos mixtos y ventas al crédito. Se muestra únicamente la lógica relevante.

Envío Rápido de Comandas

// Envío inmediato de comanda
$.post('comanda/enviar.php', {
  mesa_id,
  pedido,
  tipo: 'COCINA'
});

Optimización del flujo de comandas para cocina y delivery, reduciendo pasos y acelerando la atención al cliente.

Pagos Parciales (Yape / Plin / Efectivo)

// Registro de pagos por partes
pagos.push({
  metodo: 'YAPE',
  monto: parcial
});

Implementación de pagos mixtos permitiendo dividir el total entre Yape, Plin y efectivo, validando montos correctamente.

Ventas al Crédito

// Venta a crédito
if (tipoPago === 'CREDITO') {
  venta.estado = 'PENDIENTE';
}

Se agregó la funcionalidad de ventas al crédito, inexistente en el sistema base, permitiendo control posterior de pagos.
