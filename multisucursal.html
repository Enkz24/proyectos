Sistema de Ventas Multisucursal con Tienda Virtual

Sistema web orientado a pequeñas y medianas empresas para la gestión de ventas, almacén y clientes, extendido y adaptado a múltiples sucursales y venta online.

Descripción General

Este sistema fue desarrollado a partir de una base inicial en PHP, la cual fue modificada y ampliada para cubrir nuevos escenarios de negocio. El enfoque principal fue mejorar la flexibilidad del sistema, permitiendo operar múltiples sucursales y distintos tipos de venta.

Funcionalidades Implementadas

  • Gestión de ventas por sucursal
  • Precios diferenciados: por menor y por mayor
  • Ventas al contado y ventas al crédito
  • Módulo de tienda virtual
  • Registro de notas personalizadas por venta
  • Edición, eliminación y visualización de registros
  • Rediseño completo del ticket de venta
  • Mejoras en interfaz y experiencia de usuario
