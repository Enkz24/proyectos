Sistema POS Multisucursal

Sistema de ventas y control interno adaptado y extendido para soportar múltiples sucursales, ventas por mayor, crédito y personalización de tickets.

Descripción General

Este sistema POS fue adaptado y mejorado a partir de una base existente, incorporando funcionalidades críticas para entornos comerciales reales, como ventas al crédito, precios diferenciados y control por sucursal.

Funcionalidades Implementadas

  • Gestión de múltiples sucursales
  • Precios por mayor y por menor
  • Ventas al contado y crédito
  • Edición y eliminación de ventas
  • Notas y observaciones por venta
  • Rediseño profesional del ticket
