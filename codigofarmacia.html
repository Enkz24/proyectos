Sistema de Farmacias y Tiendas | Parte del Código

Fragmentos clave del sistema POS adaptado para farmacias y tiendas mayoristas. Se muestran las mejoras implementadas sobre control sanitario, precios y reposición.

Control de Fecha de Caducidad

// Validación de vencimiento
if (producto.fecha_vencimiento < hoy) {
  producto.estado = 'VENCIDO';
}

Se implementó control obligatorio de fechas de vencimiento para productos farmacéuticos, evitando ventas de productos caducados y cumpliendo normativa sanitaria.

Alertas Preventivas de Caducidad

// Alerta de productos por vencer
if (dias_restantes <= 30) {
  alertas.push(producto);
}

Sistema de alertas automáticas que notifica productos próximos a vencer, facilitando rotación o reposición antes de pérdidas económicas.

Precios Mayoristas y Minoristas

// Selección de precio
precio = cantidad >= minimo_mayorista
  ? precio_mayor
  : precio_unitario;

Se agregó lógica de precios por volumen, permitiendo ventas mayoristas en abarrotes y farmacias con cálculo automático según cantidad.

Compra y Reemplazo de Precios

// Actualización de precio por compra
producto.precio = nuevo_precio;
producto.costo = costo_compra;

Funcionalidad agregada para actualizar costos y precios al registrar nuevas compras, algo que el sistema base no contemplaba.
