Sistema de Integración y Visualización de Horarios Académicos

Plataforma desarrollada para unificar múltiples horarios académicos provenientes de archivos Excel, procesarlos con Python y Pandas, almacenarlos en una base de datos centralizada y visualizarlos mediante una aplicación web moderna construida con Node.js y Next.js.

Descripción del Proyecto

Este proyecto surge ante la necesidad de centralizar y normalizar horarios académicos generados en diferentes formatos Excel por distintas áreas institucionales.

Mediante un proceso automatizado en Python utilizando Pandas, los archivos son leídos, validados, transformados y unificados en una estructura estándar que posteriormente se almacena en una base de datos relacional.

Los datos consolidados son consumidos por una API desarrollada en Node.js y visualizados en una interfaz web interactiva construida con Next.js.

⚙️ Flujo de Funcionamiento

  • Carga de múltiples archivos Excel con horarios académicos
  • Lectura y limpieza de datos con Python y Pandas
  • Normalización de formatos, días, horas y aulas
  • Validación de cruces, duplicados y errores
  • Inserción de datos unificados en base de datos
  • Exposición de información mediante API Node.js
  • Visualización de horarios en aplicación web Next.js

Características Destacadas

  • Unificación automática de múltiples archivos Excel
  • Procesamiento de datos con Pandas
  • Estructuración y almacenamiento en base de datos
  • Detección de inconsistencias en horarios
  • Arquitectura desacoplada (ETL + API + Frontend)
  • Visualización clara y ordenada de horarios
  • Escalable a nuevas fuentes de datos
